# Vue

- [Vue](https://vuejs.org)
- [Nuxt](https://nuxt.com)
- [Tanstack Query](https://tanstack.com/query/latest/docs/vue/overview)
- [Tailwind CSS](https://tailwindcss.com)
- [PrimeVue (component library)](https://primevue.org)

## Setup

- Install dependencies (run in `./vue` folder):

```bash
pnpm install
```

## Development

```bash
pnpm dev
```
